# Web Microservice

## 1. Services
1.1 Web interface: `localhost:28085`  
1.2 Database microservice (connect via ZeroTier)  
1.3 HAProxy dashboard: `localhost:28404/stats`  
1.4 cAdvisor: `localhost:29090`  

## 2. Installation Requirements
2.1 [Docker](https://www.docker.com/get-started)  
2.2 [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install)  
2.3 Virtualization enabled in BIOS: Intel or AMD
2.4 [ZeroTier](https://www.zerotier.com/) for database connection  

No other dependencies are needed.
